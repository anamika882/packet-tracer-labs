# packet-tracer-labs
Network Troubleshooting &amp; Enhancement project using Cisco Packet Tracer. Fixed DHCP misconfigurations, added new PCs using VLAN 111, restored connectivity, and implemented router and switch security including port security and MAC filtering. Resulted in a stable, secure SOHO network.
# Cisco Packet Tracer Projects

This repository contains my Packet Tracer networking labs created during my CCNA and Cybersecurity studies.

## Projects Included
- VLAN Configuration and Inter-VLAN Routing
- OSPF Single Area Design
- ACL Implementation (Standard + Extended)
- NAT/PAT Configuration
- DHCP Server Implementation
- Basic Routing Lab (Static + Dynamic)

## How to Use
1. Download any `.pkt` file
2. Open with Cisco Packet Tracer 8.x or later
3. Follow instructions inside each folder

## Tools Used
- Cisco Packet Tracer 8.2
- Wireshark (for traffic monitoring)
- GNS3 (occasionally)
